Supported operations

+,-,*,/,=,>,<,>=,<=,&,!(use gamma function),^(power)

& : join two strings into one string. e.g:("114")&("514")->"114514"

Elements type

number, string(use "" to define string), boolean(true or false), NaN(usually gives Error)

Usually, true equals to 1, and false equals to 0.

Functions

Trigonometric functions:sin,cos,tan,csc,sec,cot,asin,acos,atan,acsc,asec,acot

Others:ln,log,logb(base,num),alog(alog(x) equals to 10^x),mod(a,b),floor,ceil,round

Supported input form: asin or arcsin. A function needs one Real-Number argument.

UserFunctions

Define your functions, define a functionName, and use the form like f1(x,y,z)=xy-z to define.

How to make a name of your functions?

Just use some letters and numbers to name, if you want to make a name which include math functions, you must add a "_". e.g:f1,aaa,sin_a,myfunc_1

Others you should know to make a UserFunction

SGC supported make more than two arguments for a UserFunction, and the names of arguments cannot be the same as the names of variable.
