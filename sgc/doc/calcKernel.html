支持的运算符

+,-,*,/,=,>,<,>=,<=,&,!(支持非整数输入),^(幂运算)

& : 将两个字符串拼接位一个字符串. e.g:("114")&("514")->"114514"

元素种类

数, 字符串(使用英文""包裹字符串), 布尔值(true or false), NaN(一般会显示为"?")

运算中，true相当于1，false相当于0

函数

三角函数:sin,cos,tan,csc,sec,cot,asin,acos,atan,acsc,asec,acot

Others:ln,log,logb(base,num),alog(alog(x) equals to 10^x),mod(a,b),floor,ceil,round

Supported input form: asin or arcsin. A function needs one Real-Number argument.

UserFunctions

Define your functions, define a functionName, and use the form like f1(x,y,z)=xy-z to define.

How to make a name of your functions?

Just use some letters and numbers to name, if you want to make a name which include mathematical functions, you must add a "_". e.g:f1,aaa,sin_a,myfunc_1

Others you should know to make a UserFunction

SGC supported make more than two arguments for a UserFunction, and the names of arguments cannot be the same as the names of variable.
